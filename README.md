# 🎓 EduTrade Web Portal

## 📌 Project Description

EduTrade is a web-based platform that enables students to buy, sell, and share educational resources efficiently.

---

## 🚀 Features

* 🔐 User Registration & Login
* 📤 Upload Study Materials
* 💰 Buy/Sell Notes
* 🔍 Search Functionality

---

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MongoDB

---

## 📂 Project Structure

```
frontend/
backend/
```

---

## 🌐 Live Project

👉 https://snjbedutrade.netlify.app/

---

## ▶️ How to Run Locally

1. Clone the repository
2. Navigate to frontend folder:

   ```
   npm install
   npm start
   ```
3. Navigate to backend folder:

   ```
   npm install
   npm start
   ```
